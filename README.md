# 2025-Vibe-Coding--期末專案：[我的電子衣櫥]
## 🌟 1. 專題願景 (Vibe)
目標對象：衣服整理需求的一般民眾
設計風格：奶油風、圓角設計
核心功能：將雜亂的衣櫃電子化整理，平常找不到的衣服可以輕易搜尋到，太多重複時也方便段捨離。

## 🤖 2. Gemini 協作紀錄 (Prompt Log)
請貼出你餵給 Gemini 最關鍵的一個 Prompt：
  我是個普通大學女生，現在在整理房間，請按照我的需求製作一個電子衣櫥的互動式網頁，內容包括各個類別的選單分頁，例如上衣、內搭、長褲、內衣褲等，分類進去後有每一個品項的大圖(圖片上傳)以及底下的詳細說明標籤，例如購買多久、材質、季節等，風格想要奶油風、圓角、現代感

在開發過程中，你遇到了什麼 Bug？Gemini 怎麼幫你修好的？
遇到的問題：搜尋欄只能搜尋名稱
Gemini 的建議：1.全文檢索功能：更新了 filteredItems 的過濾邏輯，現在支援對 description（備註）欄位的即時搜尋。
               2.模糊比對：採用不分大小寫的包含比對方式，符合你提到的「單一個字符合就出現」的需求。
               
 ## 📂 3. 檔案結構說明
index.html：網頁主結構
assets/：存放圖片或素材的資料夾

## 📈 4. 自我心得與反思
使用 Vibe Coding (與 AI 對話寫 Code) 的心得：
  很輕易就做出一個方便管理自己衣服的網頁，衣服很多很雜困擾我很久了，但現在只要將問題丟過去就能生成出我需要的東西。
如果可以再優化，我希望增加什麼功能？
  希望可以提醒我內衣褲過舊需要定期更新，根據購買時間關注衣服更換時間這樣。
